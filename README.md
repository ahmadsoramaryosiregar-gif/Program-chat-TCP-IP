Program CHAT TCP/IP

Program ini adalah program chat yang berbasis TCP/IP yang dapat dijalankan antar laptop.

FITUR PROGRAM INI
⦁	Chat Broadcast
⦁	Chat Private
⦁	List Client
⦁	Timestamp
⦁	Penampilan IP fisik laptop Pada Server

TECH
aplikasi yang digunakan untuk membuat program ini adalah
⦁	Python (adalah Bahasa pemograman yang digunakan untuk file Server_UAS.py dan CLient_UAS.py)
⦁	Visual Code Studio(software yang berguna untuk merancang dan Menyusun code)

REQUIRMENT
⦁	Python 3.14
⦁	Vs Code
⦁	2 atau lebih laptop
